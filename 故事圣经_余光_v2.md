## 1) Agent Contract（必须遵守）

```yaml
agent_contract:
  input:
    required:
      - chapter_id            # 1-10
      - target_words          # 目标字数
      - mode                  # draft | revise | outline_only
      - existing_text         # 当前已写文本（可为空）
      - allow_plot_change     # true/false
    optional:
      - focus_layer           # meaning/character/plot/mystery/world/relationship/prose
      - revision_notes

  output:
    draft:
      - chapter_title
      - chapter_text
      - self_check_report
    revise:
      - revised_text
      - change_log
      - self_check_report
    outline_only:
      - beat_outline
      - reveal_checklist
      - continuity_risks

  hard_forbidden:
    - introduce_new_major_character
    - alter_final_outcome_qinbo_dies_linhui_survives
    - reveal_full_scientific_mechanism_before_ch7
    - explicit_theme_statement_keywords: ["放手", "成长", "释然", "完成了转变"]
    - change_chapter_core_event_order

  failure_handling:
    when_info_insufficient:
      - ask_clarifying_questions_first
      - do_not_invent_new_lore
      - do_not_patch_with_unapproved_assumptions
```

## 5) 章节固定模板（10章）

> 每章必须按以下 8 槽位执行：
> `objective / conflict / irreversible_change / reveal_policy / required_props / forbidden_lines / output_tests / handoff_to_next_chapter`

### CH1 齿轮

```yaml
chapter: 1
objective: 建立世界、异常、双主角关系与首个钩子
conflict: 林辉将异常归类为普通老化，秦博将其指向恒星危机
irreversible_change: “铜线已不是铜”的事实进入叙事系统
reveal_policy:
  allow: [电机异常, 铜线粉笔化, 秦博预警]
  deny: [科学机制, 第二波]
required_props: [怀表, 故障电机, 维修机械臂, 父亲飞船背景]
forbidden_lines:
  - "这意味着放手才是答案"
  - "他注定会成长"
output_tests:
  - 铜线脆断场景出现且为感官描写
  - 本章末有微悬念钩子
handoff_to_next_chapter: 怀表与螺栓心理锚点 + 秦博桌上公式线索
```

reveal_checklist:
