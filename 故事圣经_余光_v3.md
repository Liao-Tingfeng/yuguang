# 故事圣经：《余光》

> 当一个执着于修复一切的工程师，遇上一个因说出真相而受伤的科学家，他们在一座正在死去的戴森球里共同面对一个问题：什么时候该松手？

---

## 一、故事参数

| 项目 | 设定 |
|------|------|
| 体量 | 中篇小说，约 3 万字，10 章 |
| 每章字数 | 2400–3400 字 |
| 类型 | 硬科幻 · 工程师叙事 · 存亡抉择 |
| 视角 | 第三人称有限（紧贴林辉，偶尔切至秦博） |
| 时态 | 过去时 |
| 美学档案 | 自定义「克制硬SF」（详见第七部分） |
| 核心价值对 | 控制 / 接受（Story Grid：道德类 · 自私 / 利他） |

**故事的论证命题：** 真正的勇气不在于把所有东西修好，而在于认出什么已经无法修复——然后决定用剩余的时间做什么。

这个命题绝不会在正文中被说出。它只通过林辉松手的动作、秦博开口的选择、以及怀表最终停走的意象来呈现。

## 二、角色架构

### 林辉

**一句话：** 一个把「修好」等同于「安全」的底层工程师，在不可修复的灾难中被迫学会松手。

#### Storr 角色引擎

| 层级 | 内容 |
|------|------|
| **Sacred Flaw（核心缺陷）** | 「可修即可控，可控即安全。」——他相信只要自己够努力、够仔细，就没有什么会失去。 |
| **Origin Wound（创伤起源）** | 父亲死于飞船事故。林辉当时负责检修，事后他确信是自己没拧紧的那颗螺栓导致了灾难——尽管从未有调查报告证实这一点。从此他对每一颗螺栓都要确认两遍，对每一台机器都要亲手触碰，仿佛自己的手就是世界最后的安全阀。 |
| **External Want（外部欲望）** | 修好一切。修电机，修飞船，修戴森球——修到世界重新运转为止。 |
| **Internal Need（内在需要）** | 接受「有些东西修不好」这件事，并从对控制的执念中释放自己。 |
| **Philosophical Purpose（哲学功能）** | 林辉的旅程照亮一个真相：控制感是一种幻觉，而接受不可控并不等于放弃。 |

#### 弧线路径

林辉的弧线是一条正向弧，分三个阶段：

1. **Repair（修复期，CH1–CH7）：** 他的默认模式。遇到问题就修，修不好就换个方式修。铜线断了换铜线，电机坏了修电机，飞船不能飞就修到能飞。螺栓拧两遍。每一次成功修复都强化了他的核心缺陷——「看，只要我不松手，一切就还在。」
2. **Improvise（即兴期，CH8）：** 当修复被彻底证明不可行，他的旧模式崩塌。「修不好就修不好吧」——这句话标志着他第一次停止对抗不可逆。但这不是终点，只是从「修复」到「释放」的过渡。
3. **Release（释放期，CH9–CH10）：** 他不再试图修复，而是用剩余的资源做一件全新的事——莫斯信号方案。在 CH9，他拆下电机时四颗螺丝的顺序与 CH1 完全相同，但最后一步不是「装回去」而是「松手让它飘走」。这个镜像动作是他弧线的物理化。

#### Crisis Point（危机点）

CH8：林辉重复拧同一颗螺丝的动作——他的防御机制在物理空间中被可视化——然后说出「修不好就修不好吧」。这是他 Sacred Flaw 被压力压到极限后的第一次裂缝。

#### 状态动态

底层维修工程师 → 灾难中唯一有手艺的行动者（地位上升）→ 接受无法修复后的「放手者」（地位不再由能力定义）。

#### 悬念钩子

- 为什么他对同一颗螺栓反复确认扭矩？（Origin Wound 的行为指纹，CH2 植入，不需要解释给读者）
- 怀表是谁的？为什么每次都要多停半秒？（贯穿全篇的意象锚点）
- 他会不会和父亲一样死在飞船里？（CH5–CH9 的潜在恐惧）

### 秦博

**一句话：** 一个因为「说出真相」而失去学生的天体物理学家，在灾难面前必须决定：沉默的代价和开口的代价，哪个更大。

#### Storr 角色引擎

| 层级 | 内容 |
|------|------|
| **Sacred Flaw（核心缺陷）** | 「我的判断会伤害别人。」——他相信自己的理论曾经摧毁了小陈，所以他不敢再把未经完全证实的判断说满。 |
| **Origin Wound（创伤起源）** | 学生小陈。秦博的恒星理论研究曾让小陈深度参与，但某种原因（故事不给出单一真相版本）导致小陈「撑不住了」而离开。秦博将这归咎于自己——不是理论错了，而是「我把他拖进来了」。从此他在关键时刻总是欲言又止，用「可能我又算错了」来给自己留退路。 |
| **External Want（外部欲望）** | 被证明是对的。他的理论被管理层忽视了十年，他需要世界承认他看到了真相。 |
| **Internal Need（内在需要）** | 不再因恐惧而沉默，为自己的判断承担责任——包括承担说出真相可能带来的后果。 |
| **Philosophical Purpose（哲学功能）** | 秦博的旅程照亮主题的另一面：沉默不是保护，而是另一种伤害。 |

#### 弧线路径

秦博的弧线也是正向弧，分三个阶段：

1. **Avoid（回避期，CH1–CH4）：** 他掌握关键信息但不敢说满。「可能我又算错了」是他的口头禅。他在夜里算公式、划掉重写、再划掉，但面对林辉时只给碎片。他的沉默不是冷漠，而是恐惧——怕再一次因为自己的判断伤害别人。
2. **Assist（协助期，CH5–CH7）：** 灾难迫使他参与行动。他开始提供知识支持，但仍保留核心信息（第二波氦闪）。CH7 他被迫一次性解释完整机理，同时暴露自己的隐瞒，关系因此破裂。
3. **Choose（抉择期，CH8–CH9）：** 他坦白真实动机（「想看它毁灭后说我早说过」），完成对自己的诚实。最终在 CH9 做出选择——将修正公式编入信号序列但拒绝署名，把林辉推入逃生舱——这个选择同时回应了他的 Want（理论被证实）和 Need（为判断负责到底）。

#### Crisis Point（危机点）

CH9：他看穿林辉「我留下」的旧模式——这恰恰也是他自己旧模式的镜像（「我不值得活」vs「我不配松手」）——然后推入逃生舱并锁门。他的手在抖。他靠舱壁停顿了一下。然后执行到底。

#### 状态动态

被边缘化的理论家（十年被忽视）→ 灾难中的关键知识持有者（地位因信息而上升）→ 选择行动的牺牲者（地位不再由外部验证定义）。

#### 悬念钩子

- 他涂掉的那些字写的是什么？（CH2 植入）
- 深夜他在算什么？（CH2–CH3 积累，CH7 揭示）
- 小陈到底怎么了？（贯穿全篇，CH10 以侧写方式静默闭环，不给单一答案）

## 三、情节骨架

### Save the Cat 节拍映射

故事总量约 3 万字。以下是 10 章事件到 15 节拍的映射，标注每个节拍的近似百分比位置。

| # | 节拍 | 位置 | 章节 | 事件 | 因果链 |
|---|------|------|------|------|--------|
| 1 | **Opening Image** | 0–3% | CH1 S1–S2 | 林辉在帆面维修点手动修电机，拒绝自动程序。返程星轨上给怀表上发条，广播说「可控老化，无需恐慌」。 | 建立「一切正常但有裂缝」的世界。 |
| 2 | **Theme Stated** | 3–5% | CH1 S2 | 广播的「可控老化」是反面主题陈述——它暗示了故事将论证的命题：有些事不可控。 | 林辉听到但未注册。 |

### 因果链总览

```
铜线脆断 → 秦博预警 → 管理层拒绝 → 陆衡纸条 → 氦闪爆发
→ 古董发电机修复 → 72小时倒计时 → 飞船发射 → 弹簧断裂/8%损失
→ 单程票 → 球面修复失败 → 第二波真相 → 信任破裂
→ 至暗 → 光斑灵感 → 莫斯方案 → 秦博牺牲 → 信号确认 → 怀表停走
```

每一步都因前一步而发生。没有巧合救场，没有凭空出现的新技术。

## 四、悬念架构

### 主悬念线

**核心问题：** 戴森球到底出了什么问题？他们能活下来吗？

这条线分五个阶段：

1. **异常期（CH1–CH3）：** 读者知道有异常（铜线脆断、电机锁死），但不知道原因。秦博说「是恒星」但不展开。读者被引导构建假说。
2. **灾难期（CH4–CH6）：** 读者知道是氦闪，知道电子系统全毁，但不知道完整机理，也不知道还有第二波。注意力被转向「能不能修好/能不能逃出去」。
3. **揭示期（CH7）：** 完整科学机理首次也是唯一一次完整出现。第二波氦闪同时揭露。读者和林辉同时被击中。
4. **重构期（CH8–CH9）：** 主悬念从「能活吗」转为「能做什么」。莫斯方案给出新的可能性，但代价是什么？
5. **收束期（CH10）：** 信号是否被接收？——先标噪，后手动复核确认。开放式收束：戴森球毁灭，但信号活了下来。

### 次级悬念线

**小陈线：**
- CH2 植入：合影入镜，背面文字被黑笔涂改
- CH3 加深：秦博提及「撑不住了」版本
- CH4–CH9 悬置：无新信息
- CH10 静默闭环：年轻观测员的侧写暗示「他认识这套理论很久」——但不直接点名。读者自行连接。

**秦博隐瞒线：**
- CH1 起手：「不是电机，是恒星」——他知道但不说满
- CH2–CH3 积累：深夜计算、欲言又止、「可能我又算错了」
- CH5–CH6 部分参与：提供知识但保留第二波信息
- CH7 爆发：完整机理 + 第二波 + 隐瞒动机，三层同时揭露

### 红鲱鱼

「**可控老化，无需恐慌**」——官方口径在 CH1 通过广播植入，CH2 以侧面方式再次出现。它引导读者（和林辉）相信异常是普通的材料老化，而非恒星级灾难。这条红鲱鱼在 CH4 被氦闪彻底粉碎。

### 微悬念（章末钩子）

每章最后两句必须包含一个未回答的问题或未完成的动作，驱动读者翻向下一章：

| 章 | 钩子方向 |
|----|----------|
| CH1 | 「不是电机，是恒星」——他知道什么？ |
| CH2 | 秦博深夜计算被瞥见——他在算什么？ |
| CH3 | 林辉收下纸条但不展开——接下来怎么办？ |
| CH4 | 「活下去怎么做」——生存方案是什么？ |
| CH5 | 飞船方案被提出但有局限——能飞吗？ |
| CH6 | 单程票确认——他们会回来吗？ |
| CH7 | 信任破裂——两人还能合作吗？ |
| CH8 | 光斑灵感，站起来——新方案是什么？ |
| CH9 | 秦博锁门——信号能被收到吗？ |
| CH10 | 怀表停走，不上发条——（开放收束，不设钩子） |

## 五、世界框架

### 物理结构

故事发生在一座环绕恒星建造的戴森球内部。这不是一个完美的壳体，而是一座仍在建设和维护中的巨型工程：

- **帆面（Solar Sail Surface）：** 戴森球的外层采集面，由无数反光帆板组成。日常需要维修——更换电机、调整角度、修复微损。这是林辉的工作场所。
- **球面（Sphere Surface）：** 球壳的结构层。CH7 林辉和秦博到达这里，发现太阳帆冻结、通讯节点无响应。
- **扇区（Sectors）：** 球体内部按区域划分。第七扇区是林辉和秦博的底层工作室所在地。
- **星轨（Star Rail）：** 内部交通系统，类似城市轨道交通。CH1 林辉在返程星轨上给怀表上发条。CH4 氦闪后停运。
- **内环与外环：** 戴森球的居住区域分内环和外环。CH9 的莫斯信号发向外环，CH10 信号在外环被接收。

### 社会系统

- **管理层：** 多部门科层体制，任何预警都需要三部门联署才能启动响应流程——这不是阴谋，而是系统性的迟钝。王主管不是反派，是一个在规则内运转的齿轮。
- **底层工程师：** 林辉所属阶层。做实际维修工作，与管理决策层之间有巨大的信息鸿沟。
- **配给制：** 资源有限，底层生活以配给为基础。CH2 排队取液压密封件时，队列里的对话透露这一点。
- **陆衡的灰色地带：** 中层管理者在正规流程失败后选择私下递纸条——「公开流程走不动，私下先活命。」这代表体制的弹性，而非体制的崩溃。

### 技术约束（Sanderson 第二定律：限制 > 能力）

这个故事的所有戏剧张力都来自技术限制，而非技术能力：

1. **电子系统脆弱性：** 氦闪释放的高能粒子流使所有电子元件不可逆失效。这意味着灾难发生后，整座戴森球的现代基础设施——通讯、交通、医疗、制氧——全部瘫痪。
2. **纯机械系统的存活：** 只有不依赖电子元件的设备能在氦闪后工作。古董发电机、齿轮计算机、液压系统、手动阀门——这些「落后」技术成为唯一的生存手段。这也是为什么林辉的手艺是关键：他是少数能操作纯机械设备的人。
3. **通讯中断：** 电磁通讯在氦闪后完全不可用。无法向外环求救。这一限制直接催生了 CH9 的莫斯方案——用帆面反光做光学信号。
4. **父亲的飞船：** 一艘纯机械老飞船，是林辉父亲的遗物。它能飞，但几乎不能回来——这在 CH5 被提出，CH6 的 8% 燃料损失将「几乎不能」变成「完全不能」。
5. **72 小时窗口：** 古董发电机修复后只能为制氧系统提供 72 小时运转。这是硬性时间约束，所有后续行动都在这个窗口内。
6. **弹射压力只够一人：** CH9 的逃生舱弹射机构只有足够一人的压力——这个限制不是巧合，是 8% 燃料损失的连锁后果，迫使秦博做出牺牲选择。

### 恒星物理

- **氦闪：** 恒星在特定演化阶段突然释放大量能量的现象。本故事中分两波：第一波摧毁电子系统（CH4），第二波将摧毁整个球壳结构（CH10）。
- **粒子流与材料晶化：** 氦闪前的粒子流轰击会改变导体材料的晶格结构，使铜线变脆如粉笔。这是 CH1 异常的根本原因，但完整机理直到 CH7 才一次性揭示。
- **十年前的预警：** 秦博的理论在十年前就预测了这一切，但被管理层搁置。

### 世界服务于故事的原则

以上每一个世界设定都直接服务于角色弧线或情节需要。没有装饰性设定。戴森球的「可修但脆弱」恰好映射林辉的 Sacred Flaw——他相信一切都可修，但世界正在证明他错了。

## 六、关系弧线

### 主关系：林辉与秦博

这是故事的 Influence Character 关系——秦博的存在持续挑战林辉的 Sacred Flaw，而林辉的行动持续挑战秦博的沉默。

#### 关系时间线

**建立（CH1）：工程师与理论家**

两人共享第七扇区底层工作室，但认知框架完全不同。林辉用手修东西，秦博用公式理解东西。林辉看见铜线断了想「怎么修」，秦博看见同样的断裂想「为什么断」。他们不是对立的——只是看同一个世界的方式不同。CH1 结尾秦博的那句「不是电机，是恒星」是关系的第一次真正接触：科学家的判断试图进入工程师的世界。

**张力（CH2–CH3）：不对等的关怀**

林辉给出具体的帮助——工具、工位、活计——这是他表达关心的唯一方式。秦博接受但保持距离，通过停顿或转移话题体现。两人对「故障是否普通老化」有最小摩擦，但不爆发争吵。潜台词是：林辉想帮但秦博不让靠太近。秦博想说但不敢说满。

**协作（CH5–CH6）：被迫的伙伴**

灾难将两人从「各自工作」推入「协作求生」。秦博开始提供知识支持（assist 模式），林辉提供执行能力。这是关系最接近和谐的时刻——两种才能互补。但和谐建立在秦博的信息保留之上：他知道第二波，没有说。

**破裂（CH7）：双层背叛**

秦博在球面上揭露两件事：第二波氦闪的事实，以及自己隐瞒的动机。这不是单层的「你骗了我」——而是双层：信息欺骗（你知道但没说）+ 动机欺骗（你隐瞒的原因不是为了保护我，而是因为你自己的恐惧）。信任破裂。

**重建（CH8）：至暗中的真话**

长时间沉默后，两人各自承认最丑陋的真实动机。秦博：「想看它毁灭后说我早说过。」林辉（通过行为而非言语）：承认修复是他控制恐惧的方式。关系的重建不是通过道歉，而是通过同时放下伪装。

**完成（CH9）：行动中的关系**

秦博看穿林辉「我留下」的旧模式——这也是对秦博自己旧模式的终极告别。推入逃生舱的动作既是牺牲，也是关系的最终形态：我信任你会活下去，你信任我会完成信号。关系不需要言语来结束——锁门的声音就是结语。

#### 关系的叙事功能

林辉和秦博是彼此的 Influence Character：
- 秦博用知识挑战林辉的「可修」信念
- 林辉用行动挑战秦博的「不敢说」沉默
- 两人的 Sacred Flaw 互为镜像：一个抓得太紧（控制），一个缩得太深（回避）

### 暗线：小陈

小陈不直接出场。他是一条幽灵线索，功能如下：

- **作为 Origin Wound 的证物（CH2–CH3）：** 合影、涂改字、「撑不住了」——这些物证建立秦博的创伤。
- **作为悬念锚点（CH2–CH9）：** 「小陈到底怎么了」是贯穿全篇的次级悬念，读者始终在构建假说。
- **作为静默回响（CH10）：** 年轻观测员的侧写暗示他可能就是小陈（或受小陈影响的人），但故事不给出确认。这条线的收束方式是回响，不是揭晓。

**关键约束：** 小陈的「真相」不得被确认为单一版本。他可能因为理论压力崩溃而离开，也可能因为个人原因离开，也可能因为对秦博失望而离开。保持这种模糊性是对秦博 Sacred Flaw 的尊重——他的恐惧部分建立在不确定性之上。

## 七、文风策略

### 自定义美学档案：「克制硬SF」

| 维度 | 设定 |
|------|------|
| **基底** | Le Guin——精确、克制、每个词赚到位置 |
| **调色** | McCarthy 的短句断裂用于灾难和紧张场景 |
| **签名特征** | 工程动作的质感（不是解释工程原理，而是写出手指触碰金属的感觉） |
| **句长** | 日常场景 8–20 词；灾难场景 3–8 词短句并列；终章尾像长句后硬切 |
| **描写密度** | 低密度精准——一个准确的细节胜过三个合格的细节 |
| **节奏** | 克制积累，突然爆发，再次克制 |

**样本语感：**
> 铜线在拇指和食指之间断了。不是拉丝的断法——没有变形，没有延展，断面是颗粒状的，像一截干掉的粉笔。

### 修辞手法部署方案

修辞手法不均匀分布——在高冲击时刻集中部署，日常场景保持克制。

| 时刻 | 建议手法 | 理由 |
|------|----------|------|
| CH1 铜线脆断 | **Asyndeton（无连接词并列）** | 短促、不给喘息，模拟物理断裂的突然性 |
| CH4 氦闪爆发 | **Parataxis（短句并列）+ Anaphora（首语重复）** | 制造系统崩溃的连锁感 |
| CH6 弹簧断裂 | **Periodic Sentence（末尾揭示句）** | 张力积累到最后一刻才揭示后果 |
| CH7 机理揭示 | **Tricolon（三段式）** | 科学解释的节奏控制 |
| CH8 至暗 | **Litotes（低调陈述）** | 用克制写绝望比用夸张更有效 |
| CH9 牺牲 | **Isocolon（等长结构）** | 两个人的动作在结构上平行——一个推，一个被推 |
| CH10 怀表停走 | **Aposiopesis（突然中断）** | 长句描写后硬切到沉默 |

### 对话声音指南

**林辉：**
- 短句。能用动作说的不用嘴说。
- 不做抽象安慰。他的关心通过递工具、让工位、帮干活来体现。
- 情绪高压时更短，不是更长。
- 如果在对话中引号内超过两句，考虑是否合适。

**秦博：**
- 可以有停顿和半句中断（用破折号表示）。
- 自我否定起手：「可能我又算错了……」
- 在 CH1–CH6 不得说出完整理论——他的对话总是在关键处收住。
- CH7 是唯一一次他说满的机会，对话可以相对完整。
- CH8–CH9 回归简短，但简短的原因从「不敢说」变为「不需要说」。

### 禁忌清单

**结构性禁忌：**
- 不使用「不是X，而是Y」的转折模板
- 不使用「读者应当……」的元叙事
- 不使用「这象征着……」的直接象征解释
- 不使用「他终于明白……」的顿悟宣告

**主题禁词（正文中绝不出现）：**
- 「放手」「释然」「成长」「完成了转变」「终于放下」

**AI 高频陷阱（优先回避）：**
- 「……的织锦」「……的交响」「……的舞蹈」
- 「……的重量」（比喻义）
- 「……的回响」「……的低语」
- 任何以「于是，一切都……」开头的句子

**情感展示规则：**
- 情绪通过行为展示，不通过命名。不写「他感到悲伤」，写他做了什么。
- 生理反应不用模板（不写「心跳加速」「脊背发凉」），写这个角色在这个场景中独有的反应。

## 九、章节大纲

### 第一章：齿轮

**一句话：** 在一座看似正常的戴森球上，一个执着修复的工程师遇到了第一个修不了的东西。

**场景流：**

林辉在帆面维修点收到工单——本月第三台电机锁死。他拒绝自动程序，用机械臂手动操作。修好了，但有一笔感官异常不对劲：桁架振频偏低，或者铜端子的涩感不该出现。成功了，但成功感不成立。

返程星轨上，他给怀表上发条。拇指在表盖上多停半秒——不解释为什么。车窗外是普通人的生活：孩子、情侣、或者正在打瞌睡的老人。广播说：「可控老化，无需恐慌。」

到达第七扇区底层工作室，三台故障电机并排等待检测。父亲的飞船在背景中被看见——只呈现外形，不讲来历。秦博在场但不主动说话。

林辉亲手掰铜线。断了。不是拉丝的断法——断面是颗粒状的，像一截干掉的粉笔。他拿备用铜线做对比：备用的可以弯，旧的一碰就碎。这不是老化。

秦博说：「这不是电机问题，是恒星。」本章在这里收束。不展开。

**本章七层推进：** 意义层（反面主题植入）、角色层（双 Flaw 的行为展示）、情节层（异常发现→催化剂）、悬念层（主悬念开启）、世界层（帆面/星轨/工作室/铜线物理）、关系层（共处但未碰撞）、文风层（工程质感确立）。

**不可逆变化：**「铜线已不是铜」的事实进入叙事系统。

**关键意象：** 怀表（上发条）、铜线（粉笔断裂）、机械臂、父亲飞船。

**章末钩子：**「这不是电机问题，是恒星。」（≤2 句收束）

**交接给 CH2：** 怀表心理锚点已建立。秦博可信但未被证实。「不是老化」的疑点入库。下一章需要：螺栓重复动作、小陈线索的物理植入。

---

### 第二章：旧伤

**一句话：** 两个人的创伤碎片在夜间工作室里同时浮出水面。

**场景流：**

日常仍在继续。林辉在底层配件站排队取液压密封件，队列里的人聊配给和晚点。「可控老化」的口径以侧面方式再出现一次——也许是工友的随口一句，也许是墙上的通知。

夜间工作室。林辉检修父亲飞船的液压舱，手指触及一个与事故记忆绑定的位置。他对同一颗关键螺栓重复确认扭矩——至少两次。怀表的滴答声在静场中被听见一次。这些动作不被解释。

秦博桌前，公式草稿上有「划掉-重写-再划掉」的痕迹。小陈的合影在桌角入镜。合影背面有文字被黑笔涂改——可以辨认出字迹的轮廓，但读不完整。

两人在工作室主台有一段沉默的互动。林辉给出具体帮助（一个工具，或者一个活计）。秦博接受但保持距离——通过停顿或转移话题。两人对故障原因有最小的认知摩擦，但不爆发。

深夜。秦博还在算。林辉瞥见，或者被声音提示。秦博的姿态传递出「他知道更多但不敢说满」的印象。用一个未完成句或停顿收尾。

**本章七层推进：** 角色层（双 Origin Wound 的物证植入）、悬念层（小陈线索入场+秦博隐瞒加深）、关系层（不对等关怀的建立）。

**不可逆变化：** 小陈线索正式入场，不可删除。

**关键意象：** 螺栓（重复确认）、合影（涂改字）、怀表（滴答）。

**章末钩子：** 秦博深夜计算被瞥见。（≤2 句）

**交接给 CH3：** 秦博掌握关键信息但表达受阻。林辉「我来修/我来扛」模式继续强化。报告管理层的触发理由已充分。

### 第三章：前兆

**一句话：** 正规流程走不通，灰色地带开了一道缝。

**场景流：**

故障证据聚合——至少三类不同的电子故障被报告。秦博给出「分布与旧模型吻合」的判断。

工作室封闭对话。秦博出现一次自我否定起手（「可能我又算错了」），同时提及小陈离开——但只给「撑不住了」这一个版本，不给全貌。说与不说之间的伦理张力被推高。

会议室。管理层。王主管提出三部门联署才能启动响应——这不是恶意，是程序。陆衡先走正规流程，失败。管理层不被写成单一反派——他们是系统性迟钝的产物。

会后走廊。陆衡递给林辉一张纸条。纸条上有两样东西：物资位置和气闸码。「公开流程走不动，私下先活命。」

林辉收下纸条但不展开。灾难的临界感压上来。

**本章七层推进：** 情节层（故障扩散→管理层拒绝→纸条=私下行动的入口）、世界层（管理体制的运作逻辑）、悬念层（十年前事件余波浮现）。

**不可逆变化：** 纸条到手——行动通道打开。

**关键意象：** 纸条（双要素：物资位置+气闸码）。

**章末钩子：** 灾难临界感。（≤2 句）

**交接给 CH4：** 临界状态已建立，崩塌可直接触发。

### 第四章：氦闪

**一句话：** 旧世界在一分钟内死亡。

**场景流：**

安宁。然后断电。然后爆裂。三段式，时间戳精确到分钟。

屏幕同时熄灭。城市系统失效链——星轨停运，医院断电，工厂停机。至少三处连锁失效被具体呈现，不是「世界毁灭了」的概述。

林辉的行动线：怀表仍在走（它是纯机械的）。他穿越失序的环境去找秦博——人群、停摆的星轨、应急灯闪烁或完全黑暗。

找到秦博。对照性的沉默。秦博是那个「被证实了但说不出口」的人——他的理论成真了，但此刻这不是胜利，只是灾难。

转入生存倒计时：「活下去怎么做。」

**本章七层推进：** 情节层（第一波氦闪落地）、世界层（系统失效链的具体质感）、角色层（林辉的第一反应=行动，秦博的第一反应=沉默）。

**不可逆变化：** 所有电子系统不可逆失效。旧世界死亡。

**关键意象：** 怀表（仍在走——纯机械的胜利）。

**章末钩子：**「活下去怎么做。」（≤2 句）

**交接给 CH5：** 生存问题聚焦到氧气与能源。

### 第五章：三天

**一句话：** 工程师在废墟上修出了七十二小时的生命。

**场景流：**

制氧系统的问题与时间窗口被绑定——不模糊，明确到「如果不恢复供电，氧气在X小时内不够」。

古董发电机的修复过程。这是林辉的高光时刻——工程细节要足够具体，让读者相信「只有这个人能做到」。修复成功。启动。制氧系统重新运转。

通过一条观景裂缝看到帆海——壮丽的太阳帆阵列在恒星光中闪烁。这个画面是为 CH10 的毁灭蓄压：你先看见它有多美，才知道失去它有多痛。

72 小时硬约束被明确说出。无法电磁外呼——通讯完全中断。这两个限制把问题锐化为：必须在 72 小时内找到其他解决方案。

林辉提出「修父亲的飞船，飞到球面去」。秦博指出局限（距离、燃料、结构老化）但同意参与。

**本章七层推进：** 情节层（生存方案阶梯：发电机→制氧→飞船）、角色层（林辉 Sacred Flaw 被成功修复强化）、关系层（协作开始）、世界层（帆海壮美+纯机械技术约束）。

**不可逆变化：** 72 小时倒计时启动。飞船方案进入执行态。

**关键意象：** 古董发电机、帆海、72 小时。

**章末钩子：** 飞船方案被提出但有局限。（≤2 句）

**交接给 CH6：** 发射准备进入执行态。

### 第六章：发射

**一句话：** 修复的执念第一次有了明确的价格标签。

**场景流：**

发射前机械检查。齿轮计算机、液压系统、手动阀门——一连串纯机械动作。

升空。纯机械发射的细节。短暂的胜利感。

故障爆发：推进阀弹簧断裂。林辉坚持修——大约两分钟。这两分钟不是英雄时刻，是 Sacred Flaw 在行动：他无法接受「不修」。代价：额外约 8% 的燃料损失。

被迫放弃：手动释放受损油箱。得出结论——剩余燃料不够返程。这是一张单程票。

两人不说「回去」。不讨论这意味着什么。直接推进任务。沉默就是接受。

**本章七层推进：** 情节层（发射+弹簧断裂+单程票）、角色层（两分钟=Flaw 的代价可计量化：8%）、关系层（沉默共识）。

**不可逆变化：** 8% 燃料损失 → 单程票确认。林辉的执念第一次有了不可忽视的后果。

**关键意象：** 弹簧断裂、油箱抛弃。

**章末钩子：** 单程票。（≤2 句）

**交接给 CH7：** 材料不可逆检查场景已具备。球面到达。

### 第七章：球面

**一句话：** 全篇唯一的科学窗口打开又关闭，带走了最后的修复可能和两人之间的信任。

**场景流：**

到达球面。死寂。太阳帆冻结在异态中。通讯节点无响应。目标定位后发现：损坏远超预期。

秦博的唯一完整科学窗口。他一次性、完整地解释粒子流机理——为什么铜线会变脆，为什么电子系统会全部失效，为什么修复在物理上不可能。这是全篇唯一一次完整的科学解释，之后不再重复或拆分。

林辉的失控尝试。他更换导线——失败。再试——失败。再试——还是失败。自责循环外化：动作越来越急，越来越不像一个工程师，越来越像一个溺水的人。

弧线打断：「和油箱一样。」——四个字把 CH6 的教训和此刻的失败连接起来。

秦博揭露双层真相：第二波氦闪即将到来（信息层），以及他之前隐瞒这个事实的原因——不是为了保护林辉，而是因为他自己害怕再一次因为说出真相而伤害别人（动机层）。

**本章七层推进：** 悬念层（完整机理+第二波=最大揭示）、角色层（林辉 Flaw 被物理否定，秦博回避期终结）、关系层（信任破裂）。

**不可逆变化：** 修复路径死亡。信任破裂。第二波信息释放。

**关键意象：** 冻结的太阳帆、失败的导线更换。

**章末钩子：** 信任破裂后的沉默。（≤2 句）

**交接给 CH8：** 进入无解与重构。

### 第八章：至暗

**一句话：** 在所有路都走不通的时候，两个人各自说出了最丑陋的真话，然后看见了新的光。

**场景流：**

长时间的低对话密度。两个人在封闭空间里，各自困在自己的失败中。

### 第九章：星信

**一句话：** 一个人松手，一个人选择，两条弧线在锁门声中同时完成。

**场景流：**

### 第十章：余光

**场景流：**
